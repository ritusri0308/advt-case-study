# 📊 Fertility Data Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive data visualization dashboard built using Power BI. The goal is to analyze fertility-related demographic data and extract meaningful insights regarding working hours, age, gender, and family characteristics.

---

## 📁 Repository Contents

* `Fertility_Dashboard.pbix` → Main Power BI dashboard file
* `Fertility2.csv` → Dataset used for analysis
* `README.md` → Project documentation

---

## 🛠 Tools Used

* Microsoft Power BI Desktop
* CSV Dataset

---

## 🚀 How to View the Dashboard

Open Locally (Recommended)

1. Download this repository (Click **Code → Download ZIP**)
2. Extract the ZIP file
3. Install Power BI Desktop (if not already installed)
4. Open the file:

   ```
   Fertility_Dashboard.pbix
   ```
5. Explore the dashboard using filters and visuals


## 📊 Dashboard Features

The dashboard includes the following visualizations:

* 📈 Line Chart → Age vs Average Work Hours
* 📊 Bar Charts → Gender & Number of Children comparison
* 📉 Histogram → Distribution of work hours
* 🔗 Scatter Plot → Relationship between age and work
* 🎛️ Filters → Gender, Children, Age

---

## 🔍 Key Insights

* Working hours increase with age
* Individuals with more children tend to work fewer hours
* Minimal variation observed across gender
* Majority of individuals fall within mid-to-high work ranges

---

## 🎯 Interactivity

* Dynamic filters (slicers) allow customized analysis
* Drill-down and tooltips enhance data exploration
* Dashboard updates in real-time based on user selection
